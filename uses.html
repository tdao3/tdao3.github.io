User Interaction

Depending on what you are making, the process used when operating the toaster
oven would be different, but the general idea should be the same. The steps are as followed:

  1. Pull open the door handle and put your food/ingredients in
  2. Close the door
  3. Decide what mode you need for your food

    4. Are you heating up food?
    Are you baking?
    Are you broiling?
    Are you toasting?

  4. Open door using the handle by pulling the door towards you
  5. Put the food into the oven, pulling out the tray if needed
  6. Choose the appropriate temperature and turn the dial
  7. Turn the knob to the appropriate mode
  8. The last step is to set the timer following the instruction near the dial

The interface of the toaster is quite simple with only a few components that need adjusting and setting. The simplicity of it makes it easier for the user to complete the steps without confusion. The only slighty confusing step would be to set the timer, but there are small text instructions near the knob that guide an inexperienced user on their first time using it. There are no buttons that need to be pressed or switches to turn on.
